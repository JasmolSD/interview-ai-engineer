# AI Engineer Interview

Build a small system that lets a user detect and label objects in images. Bonus points if you can implement a natural language approach to labeling or getting a description of images.

You're allowed to use any dataset you choose, here are some examples:

MedMNIST
NCT-CRC-HE-7K
Flickr8k
2018 Data Science Bowl (bbbc.broadinstitute.org/BBBC038) 


What to submit
A repo with your code, a short README on how to run it, and a few example conversations showing what it can and can't do. Include a brief written reflection on the hardest design decision you made and what you'd do differently with more time.

What we care about
We're less interested in benchmark numbers and more interested in how you think about the problem. How do you think about the computer vision model selection? How did you handle ambiguity? What does the system do when it's uncertain?
